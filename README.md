# Starter Web Project

This is a simple website project to show how Git and GitHub work together.

## Introduction

This example shows how different parts of the Git repository work, as well as various commmands that are used while maintaining a web project. So far we have looked at initialising a project, adding and changing files in the project, staging the changed and added files, committing them to the local Git repository, and then using the updated local repository to update the remote repository the files originally came from.

## Purpose

Sample website with plenty of files for demos

## Deployment

This is a simple project. Deployment can be to any web server, or even the local file system.
